This is a Windows Phone 8 Unit Test project for the Portable Class Library for SQLite NuGet Package v3.8.5.0

In order to build and run the project, verify that the extension SDK SQLite for Windows Phone v3.8.5, from the SQLite.org site (http://www.sqlite.org/2014/sqlite-wp80-winrt-3080500.vsix), has been properly installed.
