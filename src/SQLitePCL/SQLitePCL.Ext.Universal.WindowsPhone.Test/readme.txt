This is a Universal Windows Phone 8 Unit Test project for the Portable Class Library for SQLite v3.8.5

In order to build and run the project, verify that the extension SDK SQLite for Windows Phone 8.1 v3.8.5, from the SQLite.org site (http://www.sqlite.org/2014/sqlite-wp81-winrt-3080500.vsix), has been properly installed.
